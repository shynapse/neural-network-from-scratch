# Neural Network From Scratch

My first implementation of a neural network from scratch using Python and NumPy.

## About The Project

This project was built as part of my learning journey in artificial intelligence and machine learning.

The main goal was to understand how neural networks work internally by implementing the core concepts manually instead of using frameworks like TensorFlow or PyTorch.

Through this project, I practiced:

* Forward propagation
* Backpropagation
* Gradient descent
* Weight updates
* Activation functions

## Technologies Used

* Python
* NumPy
* Jupyter Notebook

## What I Learned

While building this project, I learned:

* How neurons process input
* How loss is calculated
* How gradients are propagated backward
* Why activation functions matter
* The basics of training a neural network

## Why This Project

I created this repository as my first step toward learning AI engineering.

This is intentionally a simple implementation focused on learning fundamentals.

Future repositories will include more advanced architectures and cleaner implementations as I continue learning.
